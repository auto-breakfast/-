## 應用情境
應用情境： 以PIC18做為整個作品的核心，以時鐘IC當計時器並在七段顯示器上顯示剩餘時間，再用手機以藍芽做為溝通橋樑傳送資料到PIC18 時間到點以後控制伺服馬達轉動角度來啟動電鍋、吐司機等等
